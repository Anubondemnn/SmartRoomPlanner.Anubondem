# TheRoom - SmartRoom
The Smart Room Planner is a Python-based program that allows users to design and organize a room by entering dimensions and adding furniture. The program calculates space usage and helps prevent overcrowding by ensuring furniture fits within the room.

# Features & Target Audience 
Prospective students who want to live by themselves after college. People moving into small apartments who need help organizing space. 

A User inputs room dimensions (length & width), then the user adds furniture with appropriate dimensions.

This allows the program to calculate:
- Total space used
- Remaining space
- Prevents adding furniture if it exceeds the room capacity
